HTML Skeleton

HTML5 Boilerplate

The basic structure required for every webpage.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

Essential Tags Directory

TagDescription
<h1> - <h6>Headings (h1 is largest)
<p>Paragraph text block
<a href="">Hyperlink/Anchor
<img src="">Image embed (self-closing)
<div>Block-level container
<span>Inline text container

Forms & Inputs

Used to collect user data.

<form action="/submit">
    <label for="name">Name:</label>
    <input type="text" id="name" required>
    
    <button type="submit">Send</button>
</form>

CSS Design

How to target elements (Selectors)

Connect your styles to HTML tags.

/* Element Selector */
p { color: blue; }

/* Class Selector (uses dot) */
.btn { border-radius: 8px; }

/* ID Selector (uses hash) */
#header { background: black; }

The Box Model

Every HTML element is a box. Understanding spacing is crucial.

PropertyDescription
marginOuter space, pushes elements away.
borderThe outline around the box.
paddingInner space, inside the box.
width/heightThe core content size.

Flexbox Layout (1D)

Align items easily in rows or columns.

.container {
    display: flex;
    justify-content: center; /* X-axis */
    align-items: center;     /* Y-axis */
    gap: 10px;               /* Space */
}

JS Logic

Variables & Data Types

Store information in memory.

let name = "Apple"; // String (changes)
const pi = 3.14;    // Number (constant)
let isTrue = true;  // Boolean

let list =["A", "B"]; // Array
let obj = { x: 1 };    // Object

DOM Manipulation

How JavaScript talks to HTML.

// Select an element
const title = document.querySelector("h1");

// Change text
title.innerText = "New Title!";

// Change styles
title.style.color = "red";

Events (Making it interactive)

const btn = document.getElementById("myBtn");

btn.addEventListener("click", () => {
    alert("Button was clicked!");
});
